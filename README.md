# Simple To-Do List App (Python)

Aplikasi To-Do List sederhana berbasis Python untuk membantu mengelola tugas harian.  
Project ini dibuat sebagai latihan dasar pemrograman Python dan cocok digunakan sebagai portofolio pemula di GitHub.

## Fitur
- Menambahkan tugas baru
- Menandai tugas sebagai selesai
- Menghapus tugas
- Penyimpanan data otomatis menggunakan file JSON
- Antarmuka GUI sederhana dengan Tkinter

## Teknologi
- Python 3
- Tkinter
- JSON

## Cara Menjalankan
1. Pastikan Python sudah terinstal
2. Clone repository ini atau download file project
3. Masuk ke folder project
4. Jalankan perintah:
   ```bash
   python todo.py
